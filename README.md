# Books by You

* Contributors
Georgios Mpovolis,Hema,Oscar Chiriboga

# Description

A simple online book store management system using JavaScript, HTML, CSS,  SQL.
It allows a user to:
* register
* search books by different criterias such as titles, authors.
* Buy books
